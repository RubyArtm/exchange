# Описание проекта
Небольшая консольная утилита на Ruby для получения и отображения курсов выбранных иностранных валют по данным Центрального банка РФ (ежедневная XML-лента XML_daily.asp). Приложение запрашивает актуальный XML на текущую дату, парсит его и выводит курсы интересующих валют в удобном человекочитаемом формате.
## Ключевые возможности
- Запрос актуального курса валют на текущую дату с официального публичного источника ЦБ РФ.
- Разбор XML-ответа и извлечение необходимых полей (код, наименование, значение курса за условную единицу).
- Фильтрация по нужным валютам по их числовым кодам (например, 840 — USD, 978 — EUR, 949 — TRY).
- Чистый консольный вывод без лишней информации.

## Архитектура и реализация
- Модульная структура:
    - Класс доменной модели Rate инкапсулирует представление одной валюты (код, наименование, курс) и отвечает за форматированный вывод.
    - Главный скрипт запуска обрабатывает сетевой запрос, парсит XML и итерируется по элементам, оставляя в выводе только нужные валюты.

- Источник данных: публичный эндпоинт ЦБ РФ XML_daily.asp с параметром date_req в формате DD/MM/YYYY.
- Сетевое взаимодействие: стандартная библиотека Net::HTTP.
- Парсинг XML: стандартная библиотека REXML::Document.
- Дата/время: стандартная библиотека Date для формирования параметра запроса.
- Отбор данных: фильтрация по массиву числовых кодов валют.

## Технологический стек
- Язык: Ruby 3.4.3.
- ОС разработки: Windows 10 (amd64).
- IDE: RubyMine 2025.1.3.
- Встроенные библиотеки: net/http, uri, rexml, date.

## Навыки, применённые при разработке
- Работа с публичными HTTP-эндпоинтами и обработкой ответов.
- Парсинг XML и навигация по DOM-дереву (REXML).
- Проектирование простых доменных моделей.
- Обработка и форматирование данных для CLI.
- Работа с датами и локализацией форматов (DD/MM/YYYY для запроса).
- Организация структурой проекта и разделение кода на модули.
- Базовая валидация/проверка наличия узлов XML перед обращением к ним.
- Подготовка проекта к запуску на чистой системе (без внешних гемов).

## Требования к окружению
- Ruby 3.4.3 или совместимая версия интерпретатора.
- Доступ в интернет для обращения к сервисам ЦБ РФ.
- Windows 10 или любая поддерживаемая ОС (при наличии Ruby).

## Установка и запуск
1. Клонируйте репозиторий локально.
2. Убедитесь, что в системе установлен Ruby требуемой версии.
3. Запустите скрипт из корня проекта:
``` bash
ruby exchange.rb
```
При успешном выполнении вы увидите список выбранных валют и их курсы на текущую дату.
## Пример вывода
``` text
Валюта: Доллар США
1 USD равен:
XX,XXXX руб.

Валюта: Евро
1 EUR равен:
YY,YYYY руб.

Валюта: Турецкая лира
1 TRY равен:
ZZ,ZZZZ руб.
```
(Значения приведены для иллюстрации; фактические курсы подтягиваются в реальном времени.)
## Как изменить набор валют
- В основном скрипте предусмотрен список числовых кодов валют для фильтрации.
- Чтобы добавить или убрать валюту, отредактируйте массив с кодами (например, 840, 978, 949).
- Список числовых кодов доступен в документации ISO 4217 и/или в выгрузке ЦБ РФ.
